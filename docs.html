JSON Composition Documentation

The Colored Noise engine allows users to programmatically define multi-voice soundscapes using a JSON schema. This allows for precise control over synthesis parameters and complex polyrhythmic modulation.

Voice Parameters

The voices array supports up to four voice objects. Each voice is an independent synthesizer with the following properties:

Property Type Range Description
color Float 0.0 - 4.0 0: Violet, 1: Blue, 2: White, 3: Pink, 4: Brown.
volume Float 0.0 - 1.0 Individual gain level.
pan Float -1.0 - 1.0 Stereo position (-1: Left, 1: Right).
loop Bool true/false Repeats the ADSR envelope.
attack Float 0.0 - 10.0 Fade-in time in seconds.

Global Settings

The global object defines master effects and signal processing applied to the mixed output of all voices.

  • pulseSpeed: Frequency of the master LFO (Hz).
  • greyNoise: Boolean. Applies an inverse A-weighting perceptual EQ.
  • satAmount: Amount of non-linear saturation (0.0 to 1.0).

Example Composition

The following JSON creates a deep, pulsing brown noise texture:

{
  "voices": [
    {
      "color": 4.0,
      "volume": 0.8,
      "loop": true,
      "attack": 2.0,
      "sustain": 0.5,
      "duration": 4.0
    }
  ],
  "global": {
    "pulseSpeed": 0.5,
    "satAmount": 0.2,
    "satMode": "warm"
  }
}
